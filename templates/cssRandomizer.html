CSS Randomizer

This is a basic CSS Randomizer using Twitter Bootstrap and Less. Basically it randomizes the background color of the page. The text color is then chosen as a complementary color with Lightness set to the 100%-lightness value of the background.

Just an idea that came to my mind to think if we can create an algorithm that basically calculates harmonic colours, then theming becomes quite a simple exercise. You choose one base background color and the rest of the color scheme is auto harmonized. Basically if you're using Twitter bootstrap there would be one base color and all the rest of the variables would be auto-calculated based on Harmonization algorithms.

Code is up on github. Will be developing this more.
